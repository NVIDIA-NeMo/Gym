# Description

# Instruction Following Resource Server

Evaluates language model responses against instruction-following criteria using Open-Instruct and IFEval.

## Usage

```bash
# Install and run
pip install -r requirements.txt
python app.py
```

## Testing

```bash
pytest tests/
```
