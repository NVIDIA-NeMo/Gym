# chemistry_direct

Nemo-Gym RL environment for verifiable chemistry question answering
(direct generation variant).

## Task Description

The agent receives a natural-language chemistry question about a molecule,
paired with a SMILES string, and must respond with a **single number** (integer
or floating-point) or a **binary 0/1 flag**.  No tool calls; the format
instruction embedded in the prompt specifies the exact output format.

Questions are drawn from a stratified sample of the [ChEMBL database][chembl]
and cover RDKit-computable molecular properties:

| Property type | Examples | Format instruction |
|---|---|---|
| `float` | MolLogP, TPSA, MolWt | Single floating-point number |
| `count` | HeavyAtomCount, RingCount | Single integer |
| `bool` | PassLipinski | 0 or 1 |
| `presence` | HasAmide, HasAromatic | 0 or 1 |
| `fragment` | NumAmide, NumAromatic | Single integer |

## Reward Signal

- **Exact-match** for integer / count / bool / presence / fragment properties:
  reward = 1.0 iff `round(predicted) == round(actual)`, else 0.0.
- **Percentile-based threshold** for float properties: reward = 1.0 iff the
  prediction's absolute error beats 95% of the prior distribution for that
  property across all ~2.85M ChEMBL molecules.  This is equivalent to the
  accuracy metric used in the offline benchmarking pipeline.

The percentile scoring requires `data/reward_stats.json` — a compact quantile
array file generated by `scripts/precompute_reward_stats.py` in the
[chemistry-benchmarking-for-lead-op][bench] repo.

## Data Generation

Data is generated from the offline benchmark pipeline:

```bash
# In chemistry-benchmarking-for-lead-op
make precompute-reward-stats
make export-nemo-gym-data EXPERIMENT_ID=full_strat025_perm

# Copy into this environment
cp workdir/nemo_gym_data/train.jsonl       resources_servers/chemistry_direct/data/
cp workdir/nemo_gym_data/validation.jsonl  resources_servers/chemistry_direct/data/
cp workdir/nemo_gym_data/example.jsonl     resources_servers/chemistry_direct/data/
cp workdir/nemo_gym_data/reward_stats.json resources_servers/chemistry_direct/data/
```

## Running

```bash
# Start all servers (resources + model)
config_paths="resources_servers/chemistry_direct/configs/chemistry_direct.yaml,\
responses_api_models/openai_model/configs/openai_model.yaml"
ng_run "+config_paths=[${config_paths}]"

# Collect verified rollouts
ng_collect_rollouts \
    +agent_name=chemistry_direct_simple_agent \
    +input_jsonl_fpath=resources_servers/chemistry_direct/data/train.jsonl \
    +output_jsonl_fpath=rollouts_out.jsonl
```

## JSONL Format

Each row:

```json
{
  "responses_create_params": {
    "input": [{"role": "user", "content": "<natural-language question>\n\n<format instruction>"}]
  },
  "expected_answer": 2.3,
  "property_type": "float",
  "property": "MolLogP",
  "chembl_id": "CHEMBL25",
  "smiles": "CC(=O)Oc1ccccc1C(=O)O"
}
```
