Evaluation tutorials are hands-on walkthroughs for running a benchmark, collecting rollouts, and reading the outputs. They assume familiarity with the basic concepts in [Evaluation](/about/concepts/evaluation) and the workflow in [Evaluation](/evaluation).

## Choose a Benchmark

Start with a benchmark that has:

- A packaged resources server and config file
- Example JSONL data for a smoke test
- A clear verifier and headline metric
- Dependencies that fit the local or cluster environment

For the first tutorial, use EvalPlus. It is a coding benchmark for Python function completion, includes a five-example smoke-test dataset, and reports both base-test and plus-test verdicts.
