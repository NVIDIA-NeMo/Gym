Reading the result

Reward is the task's native Harbor verifier score. Terminal scores may be fractional hurdle completion. SWE is generally binary. Browser reward is the Worldsims judge aggregate. A zero with no infrastructure exception is a model outcome, not a broken rollout. The final canonical set has 0 infrastructure exceptions. The 96 earlier infrastructure-failed attempts are retained separately in audit sidecars and are not scored.

For SWE, verifier-handoff recovery restores the exact agent workspace in a fresh sandbox if OpenSandbox drops the original pod. It does not change the supplied task resources, agent, verifier, or reward computation.

The browser results use the included stateless DOM-reading baseline. They measure that baseline as configured and should not be interpreted as a full Chromium computer-use result for tasks that require persistent clicking, typing, or screenshot interaction.

Token counts are cumulative across agent turns, so they measure interaction volume rather than unique prompt text. Duration includes environment preparation, agent work, and verification.
Source: NeMo Gym Harbor agent with the OpenSandbox provider. The machine-readable rollout CSV and aggregate JSON accompany this report.
